BSNL FTTH Usage

This application will try to get information from BSNL servers without login.

Sometimes BSNL server will not give usage information.

In case you are not getting result either server is not reachable or server may not be responding to request.

Please try again later or let us know if it continues.

Thank you for using the application.

If you are not getting usage data, please ensure that you are using default DNS or one of the from the below DNS Server List. If you are using Pi-Hole or any other such dns handlers, consider using secondary DNS server as one of the BSNL servers.

Reason: From August 2022, BSNL have made some changes due to which the data usage servers are not reachable via public dns servers.

Issue with BSNL, due to which data is not loading. Reason: com.elitecore.corenetvertex.core.db.exception.DBDownException: Unable to fetch profile for subscriber ID: fibernumber_sid@ftth.bsnl.in Reason: Datasource not available
